## Why Synthetic Returns Vary

1. **Single Random Path**
   - The generator creates ONE random price path
   - Like flipping a coin 100 times - you won't get exactly 50 heads
   - Expected value: 50 heads, but could be 45-55

2. **Volatility Drag**
   - Higher volatility → more variation in realized return
   - VT with 15% vol: expect ±2-3% variation
   - BTC with 80% vol: expect ±20-40% variation

3. **Monte Carlo vs Single Path**
   - Monte Carlo: Run 10,000 paths and average → gets expected return
   - Synthetic data: ONE path → varies from expected return
   - This is mathematically correct behavior!

## How to Get Real Data

1. **Get an API key:**
   - Twelve Data: https://twelvedata.com/ (Free tier available)
   - OR Alpha Vantage: https://www.alphavantage.co/ (Free)

2. **Add to `.env` file:**
   ```
   TWELVE_DATA_API_KEY=your_key_here
   ```

3. **Fetch data:**
   ```bash
   npm run update-data
   ```

4. **Verify returns in app:**
   - VT should show ~8-9%
   - BND should show ~3-4%
   - If not, data may not include dividends
